## Project Overview

Scrum Workflow is a spec-first, AI-assisted development framework for AI coding assistants. The framework defines a complete scrum lifecycle (story creation, multi-agent refinement, validation, implementation, code review, approval) as structured Markdown files interpreted by platforms like Claude Code, Cursor, and Windsurf.

## Project Structure

This project consists of 2 parts:

### create-scrum-workflow (cli-installer)

- **Type:** CLI Tool
- **Location:** `create-scrum-workflow/`
- **Tech Stack:** Node.js, Commander.js ^13.0.0, fs-extra, js-yaml, @clack/prompts
- **Entry Point:** `bin/create-scrum-workflow.js`

### scrum_workflow (framework)

- **Type:** Markdown-as-Code Library
- **Location:** `scrum_workflow/`
- **Tech Stack:** Markdown, YAML, 6 AI platforms
- **Entry Point:** `config.yaml` (commands loaded on demand)

## Cross-Part Integration

The CLI installer copies framework definitions into target projects and generates platform-specific skill shims. A SHA-256 lock file tracks installation integrity. See [Integration Architecture](./integration-architecture.md) for details.

## Getting Started

### Prerequisites

- Node.js with ES Modules support
- npm
- AI coding assistant (Claude Code recommended)

### Setup

```bash
# Clone the repository
git clone <repo-url>
cd scrum_workflow

# Install CLI dependencies
cd create-scrum-workflow && npm install && cd ..
```

### Run Locally

```bash
# Run the installer (dry run)
cd create-scrum-workflow
node bin/create-scrum-workflow.js install --yes --dry-run
```

### Run Tests

```bash
cd create-scrum-workflow
npm test
```
