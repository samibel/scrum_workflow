# Workflow Skills

This directory contains reusable workflow skills and capabilities.

## Purpose

Skills are generic, shared capabilities that can be invoked by multiple workflows. They encapsulate common patterns and best practices.

## Structure

Skills are organized by category:

```
skills/
├── guided-mode/
│   └── SKILL.md
├── synthesis/
│   └── SKILL.md
└── readiness-check/
    └── SKILL.md
```

## Planned Skills

- **guided-mode**: Clarify vague user requirements through interactive questioning
- **synthesis**: Merge multiple agent perspectives into coherent recommendations
- **readiness-check**: Validate story completeness and readiness for implementation

## Usage

Skills are invoked by workflows through the AI coding assistant's skill interface, providing consistent behavior across different commands.
